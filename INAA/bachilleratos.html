Bachilleratos

En el Instituto Nacional de Apopa, ofrecemos una variedad de programas de bachillerato diseñados para preparar a nuestros estudiantes para el éxito académico y profesional.

Descripción

En este bachillerato recibirás formación técnica necesaria para desarrollar aplicaciones de escritorio, intranet, web y dispositivos móviles.

Competencias Técnicas

  • Analizar los requerimientos de clientes para desarrollar sistemas.
  • Desarrollo de Lógica de programación.
  • Diseña soluciones informáticas y sus componentes utilizando lenguaje Unificado de Modelado (UML).
  • Administración de bases de datos relacionales utilizando SGBD como Workbench, phpMyAdmin, Postgresql, HeidiSQL, SQL Server Express.
  • Desarrollar aplicaciones web bajo arquitectura cliente-servidor utilizando los lenguajes de programación de mayor demanda en el mercado, como lo son Python, Java, C#, y frameworks como Angular, React por medio de JavaScript o TypeScript.
  • Diseñar y configurar una red virtual y sus dispositivos utilizando packet tracer para ello aplicando topologías de redes más conocidas.
  • Elaborar manuales de usuarios utilizando estándares de calidad apegados a las normas ISO e ITIL.
  • Implementar buenas prácticas de desarrollo de Software.
  • Competencias en tecnologías emergentes (Inteligencia Artificiales, Diseño y manipulación de realidad virtual, impresión 3D, video mapping).

Campos de Trabajo

  • Soporte para el mantenimiento de aplicaciones.
  • Capacitación para el desarrollo de Software.
  • Pruebas y Control de calidad de sistemas.
  • Cloud Computing (Desarrollo de Aplicaciones que se ejecutan desde la Web).
  • Desarrollo de Componentes web.
  • Herramientas para la generación de Códigos para soluciones informáticas de Software.
  • Técnico de Aseguramiento de Calidad.
  • Desarrollador de sitios y aplicaciones Web.
  • Diseñador de Sistemas.
  • Diseñador y Administrador de Base de datos.
  • Instalador y configurador de Redes LAN.
  • Operador de Sistemas Informáticos.
  • Documentador de Sistemas.
  • Ventas de soluciones de Software.

Descripción

Competencias Técnicas

  • Estudio de dispositivos electrónicos.
  • Análisis de circuitos resistivos en corriente directa.
  • Análisis de Semiconductores lineales.
  • Estudio de circuitos integrados.
  • Aplicación de circuitos digitales.
  • Análisis de dispositivos.
  • Micro controladores.
  • Control electrónico de motores.
  • Máquinas eléctricas estáticas.
